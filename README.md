# Egg Catcher Game

This is a fun and engaging Egg Catcher game developed in Java. The objective of the game is to catch falling eggs in a bowl, with different types of eggs providing different scores and effects. The game features a start screen, pause functionality, and a scoring system displayed to the player.

## Features

- **Interactive Gameplay**: Catch different types of falling eggs in a moving bowl.
- **Types of Eggs**:
  - Regular Eggs: Basic score increments.
  - Golden Eggs: Higher score increments.
  - Bomb Eggs: Game over if caught.
- **Start and Pause Functionality**: Start the game by pressing Enter and pause it the same way.
- **Score and Lives Tracking**: Real-time display of score and remaining lives.
- **Graphics and Animation**: Smooth animations and gradient-filled graphics for a visually appealing experience.
- **Sound Effects**: Background music and sound effects to enhance the gameplay experience.

## How to Play

1. **Start the Game**: Press Enter to start the game from the start screen.
2. **Control the Bowl**: Use the mouse to move the bowl horizontally and catch the falling eggs.
3. **Pause/Resume the Game**: Press Enter to pause or resume the game.
4. **Avoid Bombs**: Catch regular and golden eggs for points, but avoid bomb eggs as they will end the game.
5. **Score Tracking**: The score and remaining lives are displayed at the top of the game window.

## Setup Instructions

1. Clone the repository:
    ```sh
    git clone https://github.com/shubhusguy/Eggs-catcher-game.git
    ```
2. Navigate to the project directory:
    ```sh
    cd egg-catcher-game
    ```
3. Compile the Java files:
    ```sh
    javac Eggs.java
    ```
4. Run the game:
    ```sh
    java Eggs
    ```
